API Documentation

This documentation describes the available API endpoints for the Rural Learning Platform. All API endpoints require authentication unless specifically marked as public.

Authentication

Use JWT tokens for authentication. Include the token in the Authorization header: Authorization: Bearer your-token-here

Subjects

GET /api/subjects/

List all available subjects.

{
    "count": 10,
    "next": null,
    "previous": null,
    "results": [
        {
            "id": 1,
            "name": "Mathematics",
            "description": "...",
            "image": "..."
        }
    ]
}
GET /api/subjects/{id}/offline_content/

Get offline-available courses for a subject.

[
    {
        "id": 1,
        "title": "Basic Algebra",
        "description": "...",
        "is_offline_available": true,
        "content_url": "..."
    }
]

Courses

GET /api/courses/

List all courses with optional filters.

Query Parameters:

  • subject: Filter by subject ID
  • level: Filter by difficulty level
  • search: Search in title and description
{
    "count": 20,
    "next": "?page=2",
    "previous": null,
    "results": [
        {
            "id": 1,
            "title": "Course Title",
            "description": "...",
            "subject": 1,
            "level": "beginner",
            "is_offline_available": true
        }
    ]
}
GET /api/courses/{id}/download_content/

Get downloadable content for offline use.

{
    "id": 1,
    "title": "Course Title",
    "content_url": "...",
    "lessons": [...],
    "quizzes": [...]
}

Progress Tracking

GET /api/progress/my_progress/

Get current user's progress across all courses.

[
    {
        "course": 1,
        "lessons_completed": [1, 2, 3],
        "quizzes_completed": [1],
        "percentage": 75,
        "points": 150
    }
]
